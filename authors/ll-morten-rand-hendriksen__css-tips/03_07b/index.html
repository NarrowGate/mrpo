Understanding the line-height property

The line-height property describes the height of the line box - the virtual box around a single line of text. The default line-height value of most browsers is somewhere between 1.15 and 1.2 meaning the font size multiplied by 1.2. You'll find that in many cases, this default line height feels too tight, especially when the font size is relatively small. For larger font sizes, reducing the line height may improve the appearance and readibility of text.

Line height affects all text because when you have text, you also have a line box, and that box needs a height, and that height needs to be customizable by the designer and developer!
